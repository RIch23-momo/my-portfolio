# Richmond Odonkor – Professional Portfolio

A modern, responsive personal portfolio website built with HTML5, CSS3, JavaScript, Bootstrap 5, Font Awesome, and Google Fonts. Designed to showcase skills, services, projects, and experience in a polished, professional manner.

## Features
- Fully responsive (mobile-first)
- Dark/Light mode toggle with persistent preference
- Animated typing effect in hero
- Animated skill progress bars
- Filterable portfolio grid
- Statistics counter animation
- Testimonial carousel
- Contact form with validation
- Smooth scrolling and scroll reveal animations
- SEO friendly semantic HTML
- Optimized performance with lazy loading
